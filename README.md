# Copilot_test

## Project Overview
Copilot_test is a small, front-end-focused demo repository for a **Calculator Web App**. It includes an accessible login flow that must be completed before using the calculator.

The project contains mostly HTML and Vue, with some Go and Python present in the repository (e.g., for supporting services, utilities, or experiments).

## Usage
1. Open the app (for example, open `calculator.html` in your browser, or run the project using whatever dev server/build steps you have configured).
2. Log in with the demo credentials:
   - **Username:** `admin`
   - **Password:** `password123`
3. After logging in, use the calculator UI to perform calculations.

## Calculator Web App

`calculator.html` now includes an accessible login flow before users can use the calculator.

### Accessibility highlights

- Proper form labels and semantic sections
